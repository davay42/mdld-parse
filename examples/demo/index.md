# MD-LD Demo Examples

Quick start examples demonstrating core MD-LD patterns and features.

## Demo Examples

### 🎯 Getting Started
- **[minimal.md](./minimal.md)** - Minimal document with subject and author
- **[journal.md](./journal.md)** - Personal journal with activities, reflections, and related events

### 🍳 Everyday Examples
- **[recipe.md](./recipe.md)** - Complete recipe with ingredients, equipment, and nutritional information
- **[project.md](./project.md)** - Weekend project with tasks, resources, team, and blocking issues

### 🚀 Advanced Examples
- **[space-mission.md](./space-mission.md)** - Apollo 11 mission with crew, equipment, and objectives
- **[research.md](./research.md)** - Scientific paper with datasets, methodology, and organizations

## Key Features Demonstrated

- ✅ **Explicit Annotations** - All semantic data comes from `{...}` blocks
- ✅ **Subject Chaining** - Using `{=IRI}` and `{+IRI}` for context
- ✅ **List Items** - Explicit annotations per list item (no semantic propagation)
- ✅ **Nested Structures** - Hierarchical data organization
- ✅ **PROV-O Integration** - Complete provenance tracking
- ✅ **Fragment Subjects** - Document structuring with `{=#fragment}`
- ✅ **Reverse Relations** - Using `!p` for inverse relationships
- ✅ **Datatypes & Languages** - `^^xsd:types` and `@lang` tags

---

*Each demo is a complete, working MD-LD document that can be parsed and round-tripped safely.*
