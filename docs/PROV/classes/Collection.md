## Collection {=prov:Collection label}

> A collection is an entity that provides a structure to some constituents that must themselves be entities. These constituents are said to be member of the collections. An empty collection is a collection without members.  {prov:definition}

The prov:Collection class can be used to express the provenance of the collection itself: e.g. who maintained the collection, which members it contained as it evolved, and how it was assembled. The prov:hadMember property is used to assert membership in a collection.
